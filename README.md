# 青萍AI Skills

本项目提供青萍AI平台的各种AI技能（Skills），让用户能够便捷地使用AI能力完成各种任务。

## 项目简介

青萍AI Skills 是一个技能集合项目，每个技能都封装了特定的AI功能，提供简单易用的接口。用户可以根据需要选择使用不同的技能。

## 可用技能

### 青萍AI生图 (qingping-ai-image)

使用青萍AI生成和编辑图片的技能。

**功能特性：**
- **文生图**：根据文本描述生成新图片
- **图生图**：基于参考图片URL和文本提示词生成新图片
  - 图片编辑
  - 风格迁移
  - 内容变体生成
  - 多图融合

**使用场景：**
- 生成创意图片和艺术作品
- 图片风格转换（油画、水墨画、赛博朋克等）
- 图片内容编辑（去除物体、更换背景等）
- 多图内容融合

**详细文档：** [qingping-ai-image/SKILL.md](./qingping-ai-image/SKILL.md)

## 快速开始

### 环境要求

- Node.js 环境
- 青萍AI平台访问权限
- MCP (Model Context Protocol) 工具支持

### 安装

将本项目克隆到本地：

```bash
git clone <repository-url>
cd skills
```

### 使用方法

每个技能都有独立的配置和使用说明，请参考各技能目录下的 `SKILL.md` 文件。

#### 示例：使用青萍AI生图

**文生图示例：**
```
生成一张猫的图片
```

**图生图示例：**
```
修改这张图片，把背景换成蓝天：https://example.com/image.jpg
```

## 技能开发规范

每个技能应包含以下内容：

1. **SKILL.md** - 技能说明文档，包含：
   - 技能名称和描述
   - 功能概述
   - 触发条件
   - 执行步骤
   - 使用示例
   - 注意事项

2. **技能配置** - 包含技能的元数据：
   - name: 技能名称
   - description: 技能描述
